# Chinese_to_Minecraft_CommandBlock
用中文生成我的世界命令方块

语法是伪易语言

额外的数据类型有实体,实体集,位置,矩形空间,物品方块,生物种类

我的博客 https://www.waterwelly.com/

![图片.png](https://i.loli.net/2021/07/30/7GuIy3l8HxYWOeZ.png)

可以使用变量赋值引用和下面的函数

用给定的中文命令生成我的世界命令

所有的命令都尽量放在一起按顺序执行
